Операция	Ярлык
Найдите предыдущую команду	Стрелка вверх
Переход к началу предыдущего слова	Ctrl+стрелка влево
Удаление символов от курсора до конца командной строки	Ctrl+K
Полные команды, имена файлов и параметры	Нажатие клавиши Tab
Переход к началу командной строки	CTRL+A
Отображает список предыдущих команд	история

Местоположение	Цель
/бункер	Основные двоичные файлы команд
/ботинок	Статические файлы загрузчика, необходимые для запуска процесса загрузки.
/и так далее	Конфигурация системы для конкретного узла
/дом	Домашние каталоги пользователей
/корень	Домашний каталог для административного корневого пользователя
/lib	Основные разделяемые библиотеки и модули ядра
/мнт	Точка монтирования для временного монтирования файловой системы
/выбирать	Пакеты дополнительного прикладного программного обеспечения
/usr	Установленное программное обеспечение и общие библиотеки
/var	Переменные данные, которые также сохраняются между загрузками
/тмп	Временные файлы, доступные всем пользователям

Команда	Описание
cd ..	Возврат на один каталог назад
cd ../..	Вернуться на два каталога назад
cd или cd ~	Перейдите в домашний каталог
cd -	Перейти к предыдущему пути
